# Coolhackpad

Coolhackpad is a design with 4-switches and an OLED, it helps as a shortcut while watching yt videos or listening to music in spotify!

# BOM
4x Cherry MX Switches
4x DSA Keycaps
1x XIAO RP2040
1x Case (2 printed parts)
1x PCB
1x 0.91 inch OLED display
